# 🧠 Smart Task AI

Smart Task AI es un proyecto en desarrollo que busca combinar un gestor de tareas tradicional con capacidades de inteligencia artificial para mejorar la organización personal y la productividad.

---

## 🚀 Objetivo del proyecto

El objetivo es construir un sistema que no solo almacene tareas, sino que también pueda:

- Interpretar tareas escritas en lenguaje natural
- Priorizar tareas automáticamente
- Sugerir qué hacer en cada momento del día
- Evolucionar hacia un asistente inteligente de productividad
